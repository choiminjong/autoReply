# v1.0.0 Release Notes

## 릴리즈 정보
- 버전: v1.0.0
- 날짜: 2026-03-27
- 요약: Outlook 메일 자동 회신 시스템 최초 릴리즈 + Cursor 개발 규칙 적용

## 새 기능

### 메일 자동 회신 시스템
- Outlook Graph API 연동 (메일 수신/발신/동기화)
- 칸반보드 UI (수신 → 자동회신 → 수동처리 → 완료)
- 메일 쓰레드 타임라인 상세 패널
- WebSocket 기반 실시간 메일 동기화

### 인증 / 사용자 관리
- 회원가입 / 로그인 / 세션 관리
- CSRF 보호 미들웨어
- 암호화 유틸리티 (crypto.py)

### 관리자 기능
- 관리자 전용 페이지 (`/admin`)
- 프로젝트 관리

### Cursor 개발 규칙
- `github-upload.mdc`: "업로드해줘" 시 릴리즈 노트 작성 + git push 자동화
- `owasp-security.mdc`: OWASP Top 10 2025 기반 FastAPI 보안 가이드 상시 적용

## 마이그레이션 / 주의사항
- `.env` 파일 설정 필요: `CLIENT_ID`, `TENANT_ID`, `CLIENT_SECRET`, `SECRET_KEY`
- `setup.sh` 실행으로 초기 환경 구성
- Alembic 마이그레이션: `alembic upgrade head`
- Python 가상환경: `venv/` (git 제외)
